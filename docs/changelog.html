v2.0.0
December 2024
LATEST

Major release with web interface, multi-language plugin support, and advanced reporting system.

Added

  • Interactive web interface with Flask + SocketIO
  • Multi-tab system for managing multiple targets
  • Real-time scan progress via WebSocket
  • Notes system for target documentation
  • Secure API key manager
  • GitHub plugin installer (--add command)
  • Go plugin support with auto-compilation
  • JavaScript plugin support via Node.js
  • Multi-channel notifications (Telegram, WhatsApp, Email, Webhook)
  • Advanced reporting with filters and API key detection
  • OSINT module with deep investigation
  • Configuration manager with YAML support
  • Complete GitHub Pages documentation site

Changed

  • Refactored plugin system for multi-language support
  • Improved CLI with argparse subcommands
  • Enhanced database schema for better reporting
  • Updated documentation with comprehensive examples

Fixed

  • DNS resolution errors in container environments
  • Plugin discovery race conditions
  • Report generation encoding issues
  • Concurrent scan execution bugs
