# Penetration Test Suite - Documentation Site

Visit: https://samueldk12.github.io/penetration-test

## Structure

- index.html - Main landing page with features
- documentation.html - Complete CLI documentation
- theory.html - Pentesting theory and education

## Features

- Modern responsive design
- Dark theme with gradients
- Interactive FAQ section
- Code copy-to-clipboard
- Smooth animations
- Mobile-friendly navigation
